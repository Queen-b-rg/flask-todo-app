# Master Tasker - Flask Todo App

A simple and clean Todo Application built with **python** and the **Flask** web framework. This project was created as part of my learning with freecodecamp
##Features
* **Create tasks**: Add new items to your list.
* **Delete Tasks** : Remove tasks once they are completed
* **Update Tasks**: Edit existing tasks.
* **Persistent Storage**: Uses SQlite to save your data.
